ISCSI磁盘共享

Posted on 2018-11-05

ISCSI服务配置与管理
服务器：
1、 安装iscsi服务
Yum install iscsi* -y
Yum install targetcli -y
2、 重启服务
Systemctl restart iscsi

3、 防火墙允许通过。
Firewall-cmd –permanent –add-port=3260/tcp
Firewall-cmd –reload

4、 创建共享分区。
Fdisk /dev/sdb m n p +6G w
分完区后输入partx /dev/sdb 同步磁盘分区。

5、 进入交换界面。
在终端输入 targetcli 直接进入交换界面。

6、 修改iqn号

7、 在交换界面配置文件 。
先在机交换界面创建共享。

再创建iqn号
创建lun，如果不设置lun的序号，默认为1.

设置好服务器地址和对应服务的端口号。

设置acls列表。把客户机的iqn设置上。

在交互界面做完配置之后，退出交互界面，进入全局界面，重启iscsi服务。
Systemctl restart iscsi

客户机：
1、 安装targetcli服务。
Yum install targetcli* -y

2、 然后在终端输入man iscsiadm 查看iscsi手册。
man iscsiadmin

3、 查看命令手册之后能看到帮助，复制两条如下命令到终端命令行。
iscsiadm –mode discoverydb –type sendtargets –portal 192.168.1.10(这里是自己服务器的地址)–discover

iscsiadm –mode node –targetname iqn.2001-05.com.doe:tes(这里是服务器的iqn号)–portal 192.168.1.1:3260(这里是服务器的地址) –login

4、 之后输入fdisk -l 查看一下分区。

5、 之后再进行格式化，

mkfs -t ext4 /dev/sdb*

然后挂载磁盘
mount /dev/sdb1 /1

查看挂载情况
df -hT
完成配置。

# ISCSI磁盘共享
