pxe服务器无人值守安装系统搭建

Posted on 2018-11-05

pxe服务搭建
第一步：关闭防火墙。
二、配置ip地址，重启网络服务。如果重启失败就进入
vim /etc/sysconfig/network-script/ifcfg-eth0
把配文件里面的
onboot=no改成 onboot=yes
必须安装的服务
yum -install syslinux tftp- xinetd dhcp- -y

然后修改dhcp服务的配置文件；
在配置文件末行模式下输入： : r +要打开文件的路径
在配置文件里配好dhcp地址后，外加两行
filename “pxelinux.0”
next-server 192.168.1.100
chkconfig dhcpd on 配置dhcp服务开机自起。

然后配置tftp的配置文件
vim /etc/xinted.d/tftp
把里面的disable这一行参数改成no

然后去找linux.iso镜像文件，路径在 cd /dvd/isolinux

然后把isolinux的文件都拷贝到tftp目录中
cp /dvd/isolinux/* /var/lib/tftpboot/

然后再去找pxelinux.0引导文件
find / -name pxelinux.0
找到后将pxelinux.0引导文件拷贝到tftp目录中
cp /usr/share/syslinux/pxelinux.0 /var/lib/tftpboot/

然后创建一个目录：
mkdir /var/lib/tftpboot/pxelinux.cfg

然后将开机引导文件isolinux.cfg拷贝到这个文件里来，并且将文件夹改名为default。
cp /var/lib/tftpboot/isolinux.cfg /var/lib/tftpboot/pxelinux.cfg/default
然后重启服务，再设置开机自起。

配置http服务：

然后创建文件
mkdir /var/www/html/dvd
mount /dev/sr0 /var/www/html/dvd/
然后重启http服务，service httpd restart
然后设置开机自起: chkconfig httpd on

上面一系列步骤操作完成，然后再新建一个虚拟机在新建的虚拟机电源选项卡中选择
“打开电源时进入固件”进入之后找到Boot 将引导方式改成network网络寻址。

ISCSI磁盘共享

Posted on 2018-11-05

ISCSI服务配置与管理
服务器：
1、 安装iscsi服务
Yum install iscsi* -y
Yum install targetcli -y
2、 重启服务
Systemctl restart iscsi

3、 防火墙允许通过。
Firewall-cmd –permanent –add-port=3260/tcp
Firewall-cmd –reload

4、 创建共享分区。
Fdisk /dev/sdb m n p +6G w
分完区后输入partx /dev/sdb 同步磁盘分区。

5、 进入交换界面。
在终端输入 targetcli 直接进入交换界面。

6、 修改iqn号

7、 在交换界面配置文件 。
先在机交换界面创建共享。

再创建iqn号
创建lun，如果不设置lun的序号，默认为1.

设置好服务器地址和对应服务的端口号。

设置acls列表。把客户机的iqn设置上。

在交互界面做完配置之后，退出交互界面，进入全局界面，重启iscsi服务。
Systemctl restart iscsi

客户机：
1、 安装targetcli服务。
Yum install targetcli* -y

2、 然后在终端输入man iscsiadm 查看iscsi手册。
man iscsiadmin

3、 查看命令手册之后能看到帮助，复制两条如下命令到终端命令行。
iscsiadm –mode discoverydb –type sendtargets –portal 192.168.1.10(这里是自己服务器的地址)–discover

iscsiadm –mode node –targetname iqn.2001-05.com.doe:tes(这里是服务器的iqn号)–portal 192.168.1.1:3260(这里是服务器的地址) –login

4、 之后输入fdisk -l 查看一下分区。

5、 之后再进行格式化，

mkfs -t ext4 /dev/sdb*

然后挂载磁盘
mount /dev/sdb1 /1

查看挂载情况
df -hT
完成配置。
